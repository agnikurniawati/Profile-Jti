Visi Misi

Visi

”Program Studi Terdepan di bidang Teknik Informatika Tingkat Nasional dan Berdaya Saing Global”

Misi

1. Meningkatkan mutu, akses, dan relevansi pendidikan Politeknik di bidang Teknik Informatika untuk menghasilkan lulusan sesuai kebutuhan pekerjaan;

2. Melakukan penelitian terapan dan pengabdian masyarakat di bidang Teknik Informatika untuk mengatasi persoalan industri dan masyarakat


Laboratorium

Pemrograman

Fasilistas : 1 Server & 30 PC Workstation

Sistem Operasi : Windows Server 2008 dan WIndows 7

Aplikasi : Microsoft Office, Visio, Visual Studio.Net, Hummingbird Exceed (X windows emulator), My SQL, SQL Server dan Oracle DBMS, Java, eclipse, netbean, devC++, Rational Roose, microsoft project.

Aktifitas : - Praktikum Pemrograman Terstruktur - Praktikum algortima dan pemrograman - Praktikum pemrograman berbasis obyek - Pekerjaan tugas mata kuliah, untuk kegiatan penelitian dan pengabdian pada masyarakat.

Basis Data

Fasilistas : 1 Server & 30 PC Workstation.

Sistem Operasi : Windows Server 2008 dan Windows 7

Aplikasi Microsoft Office, MySQL, Oracle, SQL Server 2003.

Aktifitas : - Praktikum Basisdata, struktur data. - Pekerjaan tugas mata kuliah, untuk kegiatan penelitian dan pengabdian pada masyarakat.

Sistem Operasi

Fasilistas : 1 Server & 10 PC Base Apple 10 PC base windows dan 10 PC base Linux.

Sistem Operasi : Windows Server 2008 dan Windows 8, Debian dan LionX.

Aplikasi Microsoft Office, Oracle VirtualBox, VLC, Sistem Utility.

Aktifitas : - Praktikum Sistem Operasi - Praktikum Arsitektur Komputer - pemrogrmana Aplikasi Mobile - Pekerjaan tugas mata kuliah, untuk kegiatan penelitian dan pengabdian pada masyarakat.

Jaringan Komputer

Fasilistas : 1 Server & 30 PC Workstation, RouterBoard Mikrotik, Cisco Switch Managable 3065, Cisco Router 2901, Juniper Router, Juniper Firewall, RF, Antena grid, Omni, dan Yagi, wirelles router, Wimax.

Sistem Operasi : Windows Server 2008, Ubuntu Server, Redhat, Windows 7 dan Linux Ubuntu.

Aplikasi Microsoft Office, Oracle VM VirtualBox, Mikrotik OS, Packet Tracer, dan Wireshark.

Aktifitas : - Praktikum Sistem Operasi - Praktikum Jaringan Komputer 1 - Praktikum Jaringan Komputer 2 - Praktikum Komunikasi Data - Pekerjaan tugas mata kuliah, untuk kegiatan penelitian dan pengabdian pada masyarakat.
